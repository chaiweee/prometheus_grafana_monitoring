## prometheus_grafana_monitoring

Developed a custom Prometheus exporter to execute validation checks on different layers of Snowflake tables  
•	Covering batch and real time data on Snowflake  
•	Collected key quality metrics such as: Sync Status, Run Status, Pass/Fail/Warn Status, Coverage, Daily Data Throughput  
•	Visualized all metrics through Grafana dashboard  
•	Implemented alerting rules for real-time notifications on data errors
